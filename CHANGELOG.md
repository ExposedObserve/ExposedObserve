# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2025-12-26

### Added

- Initial fork from upstream project
- Complete OIDC (OpenID Connect) authentication system with client configuration, session management, and provider integration

### Changed

- Renamed project from OpenObserve to ExposedObserve throughout the codebase
- Updated copyright notices to reflect ExposedObserve fork ownership
- Updated project description, homepage, and metadata in configuration files
- Modified package keywords to reflect new project identity

### Removed

- Removed GitHub workflows and CI/CD configurations (.github/workflows/*)
- Removed GitHub issue templates and project management files
- Removed deployment configurations and Docker build files (deploy/*)
- Removed screenshot assets and documentation images
- Removed SBOM (Software Bill of Materials) files and security scan configurations
- Removed benchmarking and performance testing files
- Removed Zed editor configuration files
- Removed download scripts and legacy build artifacts
