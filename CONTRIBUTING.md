# Contributing to ExposedObserve

Thank you for considering contributing! We welcome fixes, docs, and new features.

## How to Contribute

1. Fork the repo
2. Create a branch (`feature/my-cool-thing`)
3. Commit your changes
4. Push and open a PR

## Guidelines

- Keep code clean and documented
- New features should include tests where possible
- Breaking changes require discussion
- All contributions are made under AGPL-3.0
